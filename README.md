# sunfmin/homebrew-tap

Homebrew tap for [sunfmin](https://github.com/sunfmin)'s tools.

## Install

```sh
brew install sunfmin/tap/md2json
```

Or tap first, then install by short name:

```sh
brew tap sunfmin/tap
brew install md2json
```

## Formulae

| Formula | Description |
| --- | --- |
| [`md2json`](Formula/md2json.rb) | Markdown to JSON CLI (CommonMark + GFM via goldmark, mdast-style envelope). Source: [sunfmin/md2json](https://github.com/sunfmin/md2json). |
